## When to use what

- **New users**: Read Getting Started first, then skim the User Guide.
- **Troubleshooting**: Jump to FAQ for common fixes; if unresolved, contact support from the landing page footer.
- **Feature deep dive**: See Feature Documentation for how each module works.
- **Press/marketing**: Use Marketing Assets & Media for screenshots, GIFs, and social previews.
- **Training**: Use the Video Tutorial outline to record a concise walkthrough.

## Support checklist

- Confirm you are on the latest build (refresh hard cache if needed).
- Check dashboard connectivity state (Live/Reconnecting/Offline in header).
- Verify filters are not hiding results (Dashboard/Alert History).
- If notifications are missing, confirm channels + quiet hours in Settings.
- Capture a timestamp and browser version when reporting issues.
