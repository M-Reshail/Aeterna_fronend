# Video Tutorial Outline (5–7 minutes)

Use this script to record a concise product walkthrough.

## 0:00 – 0:30 Intro

- Product value prop: “Real-time crypto alerts from 50+ sources with noise reduction.”
- Who it’s for: active traders/analysts.

## 0:30 – 1:30 Landing & Signup

- Show landing hero; CTA buttons.
- Quick registration flow and login.

## 1:30 – 3:00 Dashboard live feed

- Show incoming alerts, priority badges, unread indicator pulse.
- Open alert detail modal; mark read/dismiss; highlight toasts.
- Show filter toggle (mobile + desktop), priority chips, and search.

## 3:00 – 4:00 Alert History

- Search + filter example; refresh; export CSV action with tooltip and toast.

## 4:00 – 5:00 Settings

- Account: change password success.
- Notifications: quiet hours + channel toggles.
- Watchlist: add a token with validation, save.

## 5:00 – 6:00 Admin (if enabled)

- Mention role-based access; show the admin entry.

## 6:00 – 6:30 Performance & trust

- Mention code-splitting, real-time status indicator, uptime note.

## 6:30 – 7:00 CTA

- Invite to start free trial; note support email.

### Recording tips

- 1080p, 30fps; keep cursor movements intentional.
- Use zoom on key interactions; avoid dead air.
- Keep background audio off; add captions if possible.
